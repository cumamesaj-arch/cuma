# **App Name**: Müjde Portalı

## Core Features:

- Content Submission: Allow admin to submit data like titles, media, and social links.
- Automated Social Media Sharing: Automatically share content to predefined social media accounts based on admin settings.
- Customizable Homepage Sections: Enable the admin to create and customize multiple sections on the homepage.
- Visitor Message Box: Implement a visitor message box on the homepage.
- Media Library: Implement a Media Library for media files to be stored.
- Enhanced Admin Dashboard: The admin dashboard to manage site settings, menus, users, and content.
- AI-Powered Recommendation Tool: Generate smart recommendations of similar posts, drawing the visitor's attention to site content. This tool can select other posts to draw the user's attention.

## Style Guidelines:

- Primary color: Warm gold (#FFD700) to signify reverence and tradition.
- Background color: Soft beige (#F5F5DC) to create a neutral and comforting backdrop.
- Accent color: Deep maroon (#800000) for important links and calls to action.
- Headline font: 'Playfair' (serif) for an elegant and high-end feel.
- Body font: 'PT Sans' (sans-serif) for clear and readable content.
- Use symbolic icons related to Islamic tradition for menu items and social media links.
- Clean and organized layout with clearly defined sections for messages, videos, and other content.
